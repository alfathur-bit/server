Integrasi REST API MikroTik (RouterOS v7+)

Hubungkan web portal ini langsung ke Router MikroTik fisik Anda

Autentikasi dikirim terenkripsi melalui headers HTTPS Basic Auth

Cara Mengaktifkan REST API di MikroTik RouterOS v7:

Secara default, browser akan memblokir request ke IP lokal karena proteksi keamanan CORS / Mix-Content. Pastikan Anda menjalankan perintah berikut di New Terminal MikroTik Anda:

/ip service set www-ssl disabled=no port=443
/ip service set www disabled=no port=80

Tips: Gunakan ekstensi browser "Allow CORS" saat testing dari localhost jika browser memblokir request API langsung.
